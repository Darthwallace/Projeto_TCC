Concatenação de Strings

É a operação de combinar duas ou mais strings em uma única string. Isso é útil para criar mensagens personalizadas, exibir informações formatadas ou construir conteúdo dinâmico em um programa.

Ela ocorre de forma direta, uma após a outra, sem nenhuma condição ou iteração envolvida. Cada concatenação de strings é uma instrução independente que é executada em ordem sequencial.

//Concatenando Strings


let
novaString
=
string1 + string2 + ...;

Acima mostra uma criação de uma variável novaString recebe dois valores, string1 e string2. Não é uma soma (mesmo utilizando o símbolo '+') mas sim uma concatenação desses dois valores dessas variáveis

Um exemplo prático sobre o escopo explicado acima pode ser representado da seguinte maneira:

//Solicitando o nome

let
nome
=
"Wallace"


//Solicitando a idade

let
idade
=
26


//Exibe o resultado apos a interatividade do usuário

let
saudacao
=
"Olá, "
+ nome +
"! Você tem "
+ idade +
" anos de idade."
;
//Solicitando a idade

Console
.
log
(saudacao);

Está sendo utilizada para criar uma mensagem de saudação personalizada com base no nome e na idade. A variável saudacao contém a mensagem concatenada, que é então exibida no console. Cada operação de concatenação ocorre em uma sequência ordenada de instruções e o resultado será: Olá, João! Você tem 30 anos de idade.
