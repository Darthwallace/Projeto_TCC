While

Executa um bloco de código enquanto uma condição especificada for verdadeira. A condição é verificada antes da execução do bloco.

O "While" é uma estrutura de controle de fluxo que permite executar um bloco de código repetidamente enquanto uma condição especificada for verdadeira. Ele continua a executar o bloco de código enquanto a condição fornecida for verdadeira e é interrompido assim que a condição se torna falsa. O loop "While" é útil quando você não sabe exatamente quantas vezes deseja repetir o código, mas sabe a condição que deve ser atendida para continuar a execução.

while
(condição) {
// Bloco de código a ser repetido

}

A "condição" é avaliada antes de cada iteração do loop. Se for verdadeira, o bloco de código é executado; caso contrário, o loop é encerrado.
