Do-While

Executa um bloco de código uma vez e, em seguida, repete a execução enquanto uma condição especificada for verdadeira. A condição é verificada após a execução do bloco, garantindo que o bloco seja executado pelo menos uma vez.

O "Do-While" é uma estrutura de controle de fluxo semelhante ao loop while, mas com uma diferença fundamental: no do-while, o bloco de código é executado pelo menos uma vez, mesmo se a condição fornecida for falsa.

Após a primeira execução, a condição é verificada, e se for verdadeira, o bloco de código é executado novamente. Este tipo de loop é útil quando você precisa garantir que um bloco de código seja executado pelo menos uma vez, independentemente da condição.

Após a primeira execução, a condição é avaliada. Se for verdadeira, o bloco de código é executado novamente; caso contrário, o loop é encerrado.
