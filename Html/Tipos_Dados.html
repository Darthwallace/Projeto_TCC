String

Em JavaScript, uma string é um tipo de dado usado para representar texto. Strings são sequências de caracteres e podem ser definidas usando aspas simples ( ' ), aspas duplas ( " ) ou crases ( ` ) para template literals.

As características das strings é o seguinte: as strings não podem ser alteradas diretamente, apenas substituídas por uma nova. os métodos úteis são:

.length = Retorna o tamanho da string;

.toUpperCase() = Converte para letras maiusculas;

.toLowerCase() = Converte para letras minusculas;

.replace("antigo","novo") = Substitui parte do texto;
