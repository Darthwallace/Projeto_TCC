Alert()

Para visualizar a saída do console.log, basta abrir o console do navegador (F12 ou Ctrl+Shift+I em muitos navegadores) e acessar a aba Console. Se estiver usando Node.js, a saída aparecerá diretamente no terminal.

alert
(
"Olá, mundo"
)
//Exibe uma caixa de alerta com a mensagem "Olá, mundo!"

Outro exemplo, com uma variável:

let
nome
=
"Maria"

alert
(
"Bem-vinda, "
+nome+
"!"
)
//Exibe "Bem-vinda, Maria!"

Quando esse código é executado, uma janela pop-up aparece com a mensagem. Se precisar de alternativas menos intrusivas, você pode usar console.log() para depuração ou confirm() e prompt() para interações mais ricas.

Para saber mais sobre o assunto: Alert()
