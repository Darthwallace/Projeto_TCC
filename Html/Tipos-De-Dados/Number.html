Number

Em JavaScript, o tipo Number é usado para representar valores numéricos, incluindo números inteiros e de ponto flutuante (decimais). Diferente de algumas linguagens que diferenciam int e float, em JavaScript todos os números são do tipo Number.

Abaixo mostra um exemplo utilizando o tipo Number

let
preco
=
19.99
;
// Preco é considerado como float - Numero quebrado


let
quantidade
=
3
;
// Quantidade é considerado como int - numero inteiro


Console
.
log
(preco, quantidade);

Os tipos de number são os seguintes:

let
inteiro
=
42
;
// Numero inteiro


let
decimal
=
3.14
;
// Numero decimal (ponto flutuante)


let
negativo
=
-10
;
// Numero negativo


let
exponencial
=
2e3
;
// 2 * 10³ = 2000

O tipo Number é fundamental para cálculos matemáticos e manipulação de valores numéricos em JavaScript
