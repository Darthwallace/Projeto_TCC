Object

Em JavaScript, um Object (objeto) é uma estrutura de dados que armazena pares de chave-valor. Ele permite organizar dados de forma mais estruturada e flexível.

Os objetos são usados para representar entidades do mundo real, como um usuário, um carro, um produto, entre outros.

A forma mais comum de se utilizar o Object é da seguinte maneira:

pessoa
=

:
"Wallace"

:
26
,

:
"Desenvolvedor"

};
// Objeto com o nome pessoa


console
.
log
(pessoa);
// Exibe todo o objeto


console
.
log
(pessoa.nome);
// Exibe apenas o nome "Wallace"


console

]);
// Exibe apenas a idade "26"

Os objetos são essenciais em JavaScript, pois permitem armazenar e manipular dados de forma estruturada. Eles são a base para conceitos mais avançados como JSON, POO (Programação Orientada a Objetos) e APIs.
