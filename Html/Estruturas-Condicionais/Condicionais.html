If

Executa um bloco de código se uma condição especificada for verdadeira.

É uma instrução que permite executar um bloco de código se uma condição especificada for avaliada como verdadeira (denominada como true). O If é uma das estruturas de controle de fluxo mais básicas e fundamentais em programação, permitindo que o programa tome decisões com base em condições.

let
(condicao) {
// Bloco de codigo a ser executado se a condição for verdadeira

}

‘condicao’ é a expressão que será avaliada. Se a condição for verdadeira, o bloco de código dentro do if será executado. Caso contrário, o bloco será ignorado.

O bloco de código dentro do 'IF' pode conter uma ou mais instruções JavaScript, e essas instruções serão executadas somente se a condição especificada for avaliada como verdadeira.

Um exemplo prático sobre o escopo explicado acima pode ser representado da seguinte maneira:

let
idade
=
20


if
(idade >=
18
) {
console
.
log
(
"Você é maior de idade"
);
}
// Esta mensagem será exibida se a idade for maior ou igual a 18

Se a variável ‘idade’ for maior ou igual a 18, a mensagem "Você é maior de idade." será exibida no console. Caso contrário, se a condição não for atendida (ou seja, se a idade for menor que 18), o bloco de código dentro do ‘if’ será ignorado e nenhuma mensagem será exibida.
