Switch-Case

Avalia uma expressão e executa blocos de código com base no valor da expressão.

É uma instrução que permite testar o valor de uma expressão em relação a uma série de casos possíveis. Ele fornece uma maneira mais limpa e organizada de lidar com múltiplas condições do que vários ‘if-else if-else’ encadeados.

// E assim por diante...

default
:

// Bloco de código a ser executado se nenhum dos casos anteriores for verdadeiro

}

'expressao' é a expressão cujo valor será testado.

'valor1', 'valor2', etc., são os valores possíveis que a expressão pode ter.

Cada 'case' define um caso possível, com um valor específico que a expressão pode assumir. Se a expressão for igual a um dos valores listados, o bloco de código correspondente será executado.

O bloco 'default' é opcional e é executado se nenhum dos casos anteriores for verdadeiro.

O 'break' é usado para sair do bloco 'switch', interrompendo a execução e evitando que os casos subsequentes sejam testados.
