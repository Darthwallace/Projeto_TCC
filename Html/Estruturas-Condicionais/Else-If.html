Else-If

Permite avaliar várias condições sequencialmente.

É uma extensão do if e else, permitindo lidar com múltiplas condições de forma encadeada. Isso é útil quando há mais de duas possibilidades de resultado com base em diferentes condições.

let
(condicao1) {

// Bloco de codigo a ser executado se a condição1 for verdadeira


}
else if
(condicao2) {

// Bloco de codigo a ser executado se a condição2 for verdadeira


}
else
{

// Bloco de codigo a ser executado se nenhuma das condições anterires for verdadeira


}

'condicao1','condicao2', etc., são as expressões que serão avaliadas. Se a primeira condição for verdadeira, o bloco de código dentro do primeiro 'if' será executado. Se não, e a segunda condição for verdadeira, o bloco de código dentro do 'else if' correspondente será executado. Se nenhuma das condições for verdadeira, o bloco de código dentro do 'else' será executado.

O 'else if' permite verificar várias condições em sequência, evitando aninhar vários blocos 'if-else', o que pode tornar o código mais legível.

A nota do aluno é verificada em várias faixas, e uma mensagem é exibida com base no conceito correspondente. Se a nota for maior ou igual a 90, o aluno é aprovado com conceito A.
