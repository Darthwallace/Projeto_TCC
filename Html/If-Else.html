If-Else

Executa um bloco de código se uma condição for verdadeira e outro bloco de código se a condição for falsa.

É uma instrução que permite ao programa tomar uma decisão binária com base em uma condição: se a condição especificada for avaliada como verdadeira (true), um bloco de código é executado; caso contrário, outro bloco de código é executado. Essa estrutura permite ao programa lidar com diferentes cenários de forma alternativa.

let
(condicao) {

// Bloco de codigo a ser executado se a condição for verdadeira


}
else
{

// Bloco de codigo a ser executado se a condição for falsa


}

'condicao' é a expressão que será avaliada. Se a condição for verdadeira, o bloco de código dentro do 'if' será executado. Caso contrário, o bloco de código dentro do 'else' será executado.

O bloco de código dentro do ‘if’ ou ‘else’ pode conter uma ou mais instruções JavaScript.

Um exemplo prático sobre o escopo explicado acima pode ser representado da seguinte maneira:

let
idade
=
15


if
(idade >=
18
) {
console
.
log
(
"Você é maior de idade"
);
// Esta mensagem será exibida se a idade for maior ou igual a 18

}
else
{
console
.
log
(
"Você é menor de idade"
);
// Esta mensagem será exibida se a idade for menor que 18
}

Se a variável ‘idade’ for maior ou igual a 18, a mensagem "Você é maior de idade." será exibida no console. Caso contrário, se a condição não for atendida (ou seja, se a idade for menor que 18), o bloco de código dentro do ‘else’ será executado e a mensagem "Você é menor de idade." será exibida.
