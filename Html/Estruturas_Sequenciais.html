Estruturas Sequenciais

Instruções são executadas uma após a outra, em sequência. É a estrutura de programação mais básica e direta.

É a exibição de mensagens, valores ou resultados no console do navegador ou no console do ambiente de execução JavaScript. Esta é uma técnica comum para depurar código, verificar valores de variáveis ou comunicar informações ao desenvolvedor ou usuário.

É realizada utilizando a função console.log(), onde você passa os valores ou mensagens que deseja exibir entre parênteses. Esta função é parte do objeto console, que fornece uma variedade de métodos para saída de informações no console.

O programa usa a função Prompt() para receber os nomes do usuario.

Cada nome é armazenado em uma variável ( nome1 e nome2 ) seguindo a ordem de execução.

Por fim, os nomes são exibidos no console com a função console.log().
