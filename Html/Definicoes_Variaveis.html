Definições de Variaveis

Uma variável é um espaço na memória do computador usado para armazenar um valor. Esse valor pode ser alterado ao longo da execução do programa.

//Declarando variáveis com let


let
nome
=
"Wallace"
;
let
idade
=
26
;
let
altura
=
1.75
;

//Exibindo os valores no console


Console
.
log
(
"Nome: "
, nome);
Console
.
log
(
"Idade: "
, idade);
Console
.

(

Aqui, nome, idade e altura são variáveis que armazenam diferentes tipos de dados.

No JavaScript, podemos declarar variáveis usando três palavras-chave principais:

Var = Forma antiga, evita-se usar hoje em dia;
Let = Recomendada para valores que podem mudar;
Const = Usada para valores que não podem ser alteradas.

var
nome1
=
"Pedro"
//Pode ser redeclarado (evitar usar)


let
nome2
=
"Lucas"
//Pode mudar de valor


const
nome3
=
"Lucas"
//Não pode ser alterado

Dica- Use let para valores que podem mudar e const para valores fixos.
